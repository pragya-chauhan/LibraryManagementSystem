<h1 id="school-library-management-system">School Library Management System</h1>
<p>This is a Library Management system developed using C++ and OOP concepts. It uses file handling to store the data and provides an efficient system to manage a school library.  It is used in Sant Chinmaya Public Schoolâ€™s library. Following are some of its features:</p>
<ul>
<li>Login System</li>
<li>Search, add, modify, view and delete books</li>
<li>Issue book</li>
<li>Handling returns</li>
<li>Fine charging system</li>
</ul>

    Change quantity of book in library
  Search for books/media

    Change library quantity
